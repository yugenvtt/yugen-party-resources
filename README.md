# yugen-party-resources

A party resources manager for campaigns with macros and customization.

![yugen-party-resources](.github/assets/yugen-party-resources.gif)

## Features

- Small UI that is floating on the table to show the party resources at all times. This can be toggled on/off or opacity changed.
- GUI for players who want to track the resources instead of the UI.
- Ability to upload custom icons and use Font Awesome icons for the resources.
- Assign clicking resources to macros.
